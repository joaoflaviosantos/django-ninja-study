### 1 - Running the service:

#### Windows:

```python
pip install virtualenv

virtualenv venv

.\venv\Scripts\activate

pip install -r requeriments.txt

python manage.py runserver
```

#### Linux:

```python
pip install virtualenv

virtualenv venv

. .\venv\bin\activate

pip install -r requeriments.txt

python3 manage.py runserver
```
